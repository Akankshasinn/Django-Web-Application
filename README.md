# Django-Web-Application

A Django-based web application with product management, contact form, database integration, and Django admin panel.

## Features

- Product management system
- Contact form
- Database integration using SQLite
- Django admin panel
- Responsive web design

## Technologies Used

- Python
- Django
- HTML
- CSS
- SQLite

## How to Run

1. Clone the repository:
   git clone https://github.com/Akankhasinn/Django-Web-Application.git
2. Install Django:
   pip install django
3. Run the server:
   python manage.py runserver
4. Open:
http://127.0.0.1:8000/
